# Birko.Data.Migrations.ElasticSearch

Elasticsearch migration provider for the Birko Framework.

## Features

- Index and alias management migrations
- Mapping updates
- ElasticSearchMigration base class
- Migration settings for Elasticsearch connection

## Installation

```bash
dotnet add package Birko.Data.Migrations.ElasticSearch
```

## Dependencies

- Birko.Data.Core (AbstractModel)
- Birko.Data.Stores (Settings)
- Birko.Data.Migrations
- Birko.Data.ElasticSearch
